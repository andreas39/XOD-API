用户类

.获取用户信息

/user/profiles

访问方式：GET、POST

开放人群：任何人

参数：

参数名 类型 描述 示例
id（可选） 整数 用户ID 1
id 整数 用户ID 1
phone 字符串 电话号码 13800138000
nickname 字符串 昵称 Nick
sex 字符串 性别 男 / 女
status 字符串 用户状态 可用 / 禁用

说明及调用成功的返回值示例：

{
    "code": 0,
    "msg": "成功",
    "data": [
        {
            "id": "1",
            "phone": "13800138000",
            "nickname": "Nick",
            "sex": "男",
            "status": "可用"
        },
        {
            "id": "2",
            "phone": "18654321234",
            "nickname": "Alice",
            "sex": "女",
            "status": "禁用"
        }
    ]
}

.注册用户

/user/register

访问方式：GET、POST(建议)

开放人群：任何人

参数：

参数名 类型 描述 示例
phone 字符串 电话号码 13800138000
nickname 字符串 20字以内的昵称 Nick
sex 字符串 性别 男 / 女
password 字符串 密码 123456
smscode 整数 短信验证码 620547
无返回值

说明及调用成功的返回值示例：

需要先请求短信验证码接口向手机发送验证码，再请求本接口进行注册。

{
    "code": 0,
    "msg": "成功",
    "data": null
}
